== Installation ==

This section describes how to install the plugin and get it working.
Install From WordPress Admin Panel:

1. Login to your WordPress Admin Area
2. Go to Plugins -> Add New
3. Type "WP Social Ninja" into the Search and hit Enter.
4. Find this plugin Click "install now"
5. Activate The Plugin
6. Use [wp_social_ninja id="template_id" platform="platform_name"] Shortcode on your next page or post.

= Manual Installation =
1. Download the plugin from WordPress.org repository
2. On your WordPress admin dashboard, go to Plugins -> Add New -> Upload Plugin
3. Upload the downloaded plugin file (wp-social-reviews.zip) and click Install Now
4. Activate "WP Social Ninja" from your Plugins page.
5. Use WP Social Ninja on your site form the dashboard.


== Frequently Asked Questions ==

= Does the plugin update the social feed in real-time? =
Yes, the WP Social Ninja updates the feed in real-time and fetches the latest posts automatically.

= How to customize a carousel setting? =
To do that, select the Carousel layout in the layouts types section from the editor panel. As carousel settings pop up, you are good to go.

= Can I connect with multiple Facebook Accounts? =
Yes. You can connect multiple accounts to fetch the Facebook feed on your website.

= Can I embed Tweets on a website by Hashtags or Mentions? =
Yes, WP Social Ninja allows you to fetch feeds from hashtags or mentions as a source for your feed.

= How can I get my YouTube channel ID? =
You can display videos from any channel using the unique YouTube channel ID. You can find this by viewing the channel page on YouTube and looking at the URL’s end. For example, UCideYfnGx9e07hYWf0Hz7sw.

= Can I display videos on fullscreen? =
Set popup mode and your videos will be displayed in the foreground with views, likes/dislikes, comments, descriptions, and subscribe button. Pop-up settings have 13 elements available that you can switch on or off.

= Can I connect with multiple Instagram Accounts? =
Yes. You can connect multiple accounts to fetch Instagram feeds on your website.

= Can I fetch Instagram feeds based on User Account/ Hashtags campaigns? =
Yes, you can import Social Feeds from an individual Users Account, and it’s free. Moreover, WP Social Ninja allows you to fetch hashtag campaigns on your website. However, for that, you need the premium version.

= How can I obtain API Key and Place ID? =
You don’t have to be a tech-savvy person in order to obtain the API Key & Place ID. Our team has made this plugin, especially for the non-techies. So all you have to click on the gear icon to get the API Key & Place ID

= Can I hide specific reviews or ratings on my website? =
Yes, you can. WP Social Ninja is a fully-featured review plugin with unique features. To hide a particular review on your website, you need the Pro version. Go to the Filter by Reviews Title and customize your social reviews the way you want.

= Are there pre-built templates and layout types included in the plugin? =
Yes, WP Social Ninja offers 9 pre-built available templates and three Layout Types for social reviews altogether. However, for different Layout Types, you need to upgrade to the Premium version.

= How many social reviews can be displayed on the website? =
You can display an unlimited number of reviews on your website from the Number of Reviews section.

= Can I schedule the social chat widgets? =
Yes, you can set the time schedule for when you are offline.

= Where can I display my chat widgets? =
WP Social Ninja allows you to display chat widgets in all the pages. However, you can customize and select specific pages.

= Can I customize my chat box? =
Yes, WP Social Ninja has a comprehensive settings panel for every platform. You can adjust your colors, font size, welcome message, and other settings within a few clicks.
